# nodejs example

## How to use

If you don't wanna use `wasm-pack` you need replace `@qqmee/sdk-account-nodejs` value `file:..` with `^0.1.2` before you continue.

Example

```json
{
  "dependencies": {
    "@qqmee/sdk-account-nodejs": "^0.1.2"
  }
}
```

Install deps

```bash
npm i
```

Run

```bash
node account.mjs
node address.mjs
```
